Titanic SQL Analysis

This project demonstrates basic SQL skills using the Titanic dataset.

Skills used:
- SELECT
- WHERE
- GROUP BY
- Aggregation functions
- Data filtering

Key analysis:
- Survival rate by gender
- Passenger distribution by class
- Average age and fare

This project shows ability to extract and summarise data using SQL.
